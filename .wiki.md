# Project Summary
The Jasmin SMPP SMS Gateway web interface is a robust platform that enables businesses to efficiently manage SMS messaging services via the SMPP protocol. It provides a seamless communication workflow with tools for SMS management, monitoring, and reporting, all accessible through an intuitive web dashboard. The project is now publicly available as 'jasmin-smpp-dashboard', allowing users to leverage its capabilities for enhanced messaging operations.

# Project Module Description
The project consists of several functional modules:
- **Web Dashboard**: A comprehensive interface for users to monitor and manage SMS gateways.
- **User Management**: Components for user authentication and role management.
- **Connector Management**: Tools to manage SMS connectors and their statuses.
- **Statistics and Reporting**: Visual representation of SMS statistics and performance metrics.
- **Error Handling and Loading States**: Mechanisms for managing errors and user feedback during data retrieval.

# Technology Stack
- **Frontend**: React, Vite, Tailwind CSS
- **State Management**: Redux
- **Styling**: PostCSS, ESLint
- **Build Tool**: Vite
- **API Handling**: Axios with interceptors for error management

# Usage
1. Install dependencies using `npm install`.
2. Build the project with `npm run build`.
3. Run the application using `npm run dev`.
