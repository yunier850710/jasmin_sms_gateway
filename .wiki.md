# Project Summary
The Jasmin SMPP SMS Gateway web interface is a powerful platform designed for businesses to manage SMS messaging services using the SMPP protocol. It enhances communication workflows by providing tools for efficient SMS management, monitoring, and reporting through an intuitive web dashboard. The project is now publicly available as 'jasmin-smpp-dashboard'.

# Project Module Description
The project consists of several functional modules:
- **Web Dashboard**: A comprehensive interface for users to monitor and manage SMS gateways.
- **User Management**: Forms and components for user authentication and role management.
- **Connector Management**: Tools to manage SMS connectors and their statuses.
- **Statistics and Reporting**: Visual representation of SMS statistics and performance metrics.
- **Error Handling and Loading States**: Improved mechanisms for managing errors and user feedback during data retrieval.

# Technology Stack
- **Frontend**: React, Vite, Tailwind CSS
- **State Management**: Redux
- **Styling**: PostCSS, ESLint
- **Build Tool**: Vite
- **API Handling**: Axios with interceptors for error management

# Usage
1. Install dependencies using `npm install`.
2. Build the project with `npm run build`.
3. Run the application using `npm run dev`.
