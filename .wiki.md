# Project Summary
The Jasmin SMPP SMS Gateway web interface is designed to provide a user-friendly platform for managing SMS messaging services using the SMPP protocol. The project aims to streamline communication processes for businesses by enabling efficient SMS management, monitoring, and reporting through a web dashboard interface.

# Project Module Description
The project includes various functional modules:
- **Web Dashboard**: A comprehensive interface for users to monitor and manage SMS gateways.
- **User Management**: Forms and components for user authentication and role management.
- **Connector Management**: Tools to manage SMS connectors and their statuses.
- **Statistics and Reporting**: Visual representation of SMS statistics and performance metrics.

# Usage
1. Install dependencies using `npm install`.
2. Build the project with `npm run build`.
3. Run the application using `npm run dev`.
